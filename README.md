# Tic-Tac-Toe

A browser-based Tic-Tac-Toe game built with **JavaScript, HTML, and CSS** as part of The Odin Project curriculum.

The project focuses on practicing:
- Factory functions and module pattern
- Separation of game logic and UI
- DOM manipulation and event handling
- Game state management
- Basic AI (bot opponent)

---

## Features

- Player vs Player mode
- Player vs Bot mode
- Score tracking with multi-round gameplay
- Win and draw detection
- Modal-based player setup
- Responsive UI
